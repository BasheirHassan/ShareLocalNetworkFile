## 📋 نظرة عامة

**Share Local Network File** هو تطبيق ويب مفتوح المصدر يتيح لك مشاركة الملفات بسهولة وسرعة بين الأجهزة المتصلة بنفس الشبكة المحلية. يعمل التطبيق كخادم محلي يسمح للمستخدمين برفع وتنزيل ومعاينة الملفات مباشرة عبر المتصفح دون الحاجة إلى خدمات سحابية خارجية أو إنترنت.

### 🌟 لماذا تستخدم هذا التطبيق؟

- **خصوصية وأمان**: جميع الملفات تبقى داخل شبكتك المحلية ولا تخرج إلى الإنترنت
- **سرعة عالية**: نقل الملفات بسرعة عالية داخل الشبكة المحلية دون قيود الإنترنت
- **سهولة الاستخدام**: واجهة بسيطة وسهلة الاستخدام بدعم كامل للغة العربية
- **بدون تسجيل**: لا حاجة لإنشاء حسابات أو تسجيل دخول

## ✨ الميزات الرئيسية

### 📤 مشاركة الملفات
- **رفع متعدد للملفات**: إمكانية رفع عدة ملفات دفعة واحدة
- **السحب والإفلات**: دعم خاصية سحب الملفات وإفلاتها مباشرة في منطقة الرفع
- **ملفات كبيرة**: دعم رفع ملفات كبيرة الحجم (حتى 1 جيجابايت افتراضياً)
- **مشاركة عامة وخاصة**: إمكانية مشاركة الملفات مع جميع المستخدمين أو مع مستخدم محدد

### 👁️ معاينة الملفات
- **معاينة مباشرة**: معاينة أنواع مختلفة من الملفات مباشرة في المتصفح:
  - **صور**: JPG, PNG, GIF, SVG, WebP
  - **فيديو**: MP4, WebM, MKV
  - **صوت**: MP3, WAV, OGG
  - **مستندات**: PDF, TXT, Markdown
  - **أكواد**: HTML, CSS, JavaScript, JSON, XML

### 🔄 تحديثات مباشرة
- **تحديثات فورية**: تحديث قائمة الملفات والمستخدمين في الوقت الفعلي عبر Socket.IO
- **إشعارات**: إشعارات فورية عند رفع ملفات جديدة أو انضمام مستخدمين جدد

### 👥 إدارة المستخدمين
- **قائمة المستخدمين**: عرض قائمة بالمستخدمين المتصلين حالياً
- **تعديل الأسماء**: إمكانية تعديل اسم المستخدم وحفظه في التخزين المحلي
- **معلومات الأجهزة**: عرض معلومات عن أجهزة المستخدمين (نوع الجهاز، نظام التشغيل، المتصفح)

### 🧹 إدارة الملفات
- **تنظيف تلقائي**: حذف الملفات القديمة تلقائياً لتوفير مساحة التخزين
- **تنظيف يدوي**: إمكانية تنظيف الملفات القديمة يدوياً
- **إعدادات قابلة للتخصيص**: تخصيص فترة الاحتفاظ بالملفات وجدولة التنظيف

### 📱 تجربة مستخدم متميزة
- **تصميم متجاوب**: يعمل على جميع أحجام الشاشات (أجهزة الكمبيوتر، الأجهزة اللوحية، الهواتف الذكية)
- **مشاركة عبر QR Code**: مشاركة رابط التطبيق مع الأجهزة الأخرى عبر رمز QR
- **واجهة عربية**: دعم كامل للغة العربية في جميع أجزاء التطبيق
- **تصميم بسيط**: واجهة نظيفة وبسيطة بدون تأثيرات بصرية معقدة

## 🔧 متطلبات النظام

- **Node.js**: الإصدار 14 أو أحدث
- **متصفح ويب حديث**: Chrome, Firefox, Safari, Edge
- **شبكة محلية**: جميع الأجهزة يجب أن تكون متصلة بنفس الشبكة المحلية

## ⚙️ التثبيت والإعداد

### التثبيت

1. قم بتنزيل أو استنساخ هذا المستودع:

```bash
git clone https://github.com/BasheirHassan/ShareLocalNetworkFile.git
cd ShareLocalNetworkFile
```

2. قم بتثبيت التبعيات:

```bash
npm install
```

3. قم بتشغيل التطبيق:

```bash
npm start
```

4. افتح المتصفح وانتقل إلى العنوان المعروض في نافذة الطرفية (عادةً `http://localhost:3001`).

### الإعدادات المتقدمة

يمكنك تخصيص إعدادات التطبيق عن طريق إنشاء ملف `.env` في جذر المشروع:

```bash
# المنفذ الذي سيعمل عليه التطبيق
PORT=3001

# عمر الملف بالساعات قبل الحذف (افتراضي: 24 ساعة)
MAX_FILE_AGE_HOURS=24

# تفعيل الحذف التلقائي عند بدء التطبيق (true/false)
AUTO_CLEANUP_ON_START=true

# تفعيل الحذف التلقائي الدوري (true/false)
ENABLE_PERIODIC_CLEANUP=true

# فترة الحذف الدوري بالساعات (افتراضي: كل 6 ساعات)
CLEANUP_INTERVAL_HOURS=6

# واجهة الشبكة المحددة (اختياري)
NETWORK_INTERFACE=
```

## 📝 دليل الاستخدام

### رفع وتنزيل الملفات

#### رفع ملف
1. انقر على زر "تصفح الملفات" أو اسحب الملفات وأفلتها في منطقة الرفع
2. انقر على زر "رفع الملفات"
3. انتظر حتى اكتمال عملية الرفع (سيظهر شريط تقدم)
4. سيظهر الملف تلقائياً في قائمة "الملفات المشتركة العامة"

#### تنزيل ملف
1. ابحث عن الملف المطلوب في قائمة الملفات المشتركة
2. انقر على أيقونة التنزيل ⬇️ بجانب الملف
3. سيبدأ تنزيل الملف تلقائياً إلى جهازك

#### معاينة ملف
1. انقر على أيقونة المعاينة 👁️ بجانب الملف
2. سيفتح نافذة معاينة تعرض محتوى الملف (إذا كان نوع الملف مدعوماً)
3. يمكنك تنزيل الملف مباشرة من نافذة المعاينة

### مشاركة التطبيق مع أجهزة أخرى

1. تأكد من أن الأجهزة الأخرى متصلة بنفس الشبكة المحلية
2. انظر إلى معلومات الشبكة المعروضة في أعلى الصفحة للحصول على رابط الوصول
3. يمكن للأجهزة الأخرى الوصول إلى التطبيق عبر عنوان IP الخاص بجهازك والمنفذ (مثال: `http://192.168.1.100:3001`)
4. يمكن أيضاً مسح رمز QR المعروض في التطبيق للوصول السريع

### إرسال ملف لمستخدم محدد (مشاركة خاصة)

1. انتقل إلى قائمة "المستخدمون المتصلون"
2. انقر على زر "إرسال ملف" بجانب اسم المستخدم المطلوب
3. اختر الملف الذي تريد إرساله
4. انقر على زر "إرسال الملف"
5. سيتلقى المستخدم المستهدف إشعاراً بالملف المرسل
6. سيظهر الملف في قسم "الملفات الخاصة" لكل من المرسل والمستقبل

### تعديل اسم المستخدم

1. انقر على حقل اسم المستخدم في الشريط العلوي
2. أدخل الاسم الجديد (يجب أن يكون بين 2-30 حرفاً)
3. انقر على زر الحفظ ✓
4. سيتم حفظ الاسم الجديد وإشعار جميع المستخدمين بالتغيير
5. سيتم حفظ الاسم في التخزين المحلي للمتصفح لاستخدامه في الزيارات القادمة

### تنظيف الملفات القديمة

1. انقر على زر "تنظيف" في أعلى قائمة الملفات المشتركة
2. سيتم حذف جميع الملفات التي تجاوزت الفترة المحددة (24 ساعة افتراضياً)
3. ستظهر رسالة تأكيد توضح عدد الملفات التي تم حذفها وحجمها الإجمالي

## 🛠️ التخصيص المتقدم

### تغيير المنفذ

يمكنك تغيير المنفذ الافتراضي (3001) عن طريق تعيين متغير البيئة `PORT`:

```bash
PORT=8080 npm start
```

### تخصيص حجم الملف الأقصى

يمكنك تعديل الحد الأقصى لحجم الملف في ملف `server.js`:

```javascript
app.use(fileUpload({
  createParentPath: true,
  limits: { fileSize: 1024 * 1024 * 1024 } // 1 جيجابايت
}));
```

### تخصيص إعدادات تنظيف الملفات

يمكنك تخصيص إعدادات التنظيف في ملف `.env`:

```bash
# عمر الملف بالساعات قبل الحذف (افتراضي: 24 ساعة)
MAX_FILE_AGE_HOURS=48

# تفعيل الحذف التلقائي عند بدء التطبيق (true/false)
AUTO_CLEANUP_ON_START=true

# تفعيل الحذف التلقائي الدوري (true/false)
ENABLE_PERIODIC_CLEANUP=true

# فترة الحذف الدوري بالساعات (افتراضي: كل 6 ساعات)
CLEANUP_INTERVAL_HOURS=12
```



## 🔒 الأمان والخصوصية

- **شبكة محلية فقط**: جميع البيانات تبقى داخل شبكتك المحلية ولا تخرج إلى الإنترنت
- **لا يوجد تخزين دائم**: الملفات تُحذف تلقائياً بعد فترة محددة
- **لا يوجد تتبع**: لا يتم جمع أي بيانات شخصية أو إحصائيات استخدام
